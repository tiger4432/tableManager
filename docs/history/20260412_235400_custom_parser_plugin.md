# 프로젝트 이력: 커스텀 스크립트 기반 플러그인 파서 시스템 (2026-04-12)

## 1. 개요
기존의 정형 Regex 파서로는 대응이 불가능한 특수한 데이터 포맷(2D 매트릭스 등)을 처리하기 위해, 사용자가 직접 작성한 Python 스크립트를 파이프라인에 주입하는 플러그인 시스템을 구축함.

## 2. 상세 작업 내용

### 2.1 워크스페이스 스크립트 감지 로직
- `DirectoryWatcher`가 각 테이블 워크스페이스 하위의 `scripts/custom_parser.py` 파일을 자동 탐색하도록 리팩토링함.
- 해당 파일이 존재할 경우, `AdvancedIngester`(Regex 기반)를 건너뛰고 커스텀 스크립트를 최우선으로 실행함.

### 2.2 동적 모듈 로드 및 실행 (`importlib`)
- `importlib.util`을 사용하여 파서를 런타임에 동적으로 로드하고 `parse_file(file_path)` 함수를 호출함.
- 사용자는 복잡한 서버 로직을 알 필요 없이, 파일 경로를 받아 `list[dict]` (로우 리스트)를 반환하기만 하면 데이터 적재가 완료됨.

### 2.3 2D-to-XYZ 레퍼런스 템플릿 제공
- 사용자 요청에 따라 인접한 비정형 데이터를 좌표계 테이블로 변환하는 예제 스크립트를 제공하여 구현 난이도를 낮춤.

## 3. 최종 결과
- 사용자가 독자적인 로직을 담은 스크립트를 배치하는 것만으로, 어떠한 형태의 로그 파일도 `assyManager`의 정형 테이블로 즉시 실시간 통합이 가능해짐.
