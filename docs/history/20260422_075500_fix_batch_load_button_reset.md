# 2026-04-22: 1K 로드 버튼 복구 로직 최적화

## 1. 개요
전체 데이터를 모두 로드하여 더 이상 가져올 데이터가 없을 때, 1K 로드 버튼이 ⏳ Loading 상태로 고정되는 현상을 해결함.

## 2. 변경 사항
- **`ApiLazyTableModel.fetch_batch` 수정**: 
    - `increment <= 0` (남은 데이터 없음) 조건 탐지 시 즉시 `batch_fetch_finished` 시그널을 발생시키도록 수정.
    - 이를 통해 `MainWindow`와 연결된 `FilterToolBar`의 버튼이 즉시 활성화 상태로 복구됨.

## 3. 검증 결과
- 모든 행이 로드된 후 1K 로드 버튼 클릭 시, 1초 미만으로 버튼이 복구되며 불필요한 대기 시간이 사라짐을 확인.
