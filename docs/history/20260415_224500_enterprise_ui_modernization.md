# 2026-04-15 22:45 | Enterprise UI Modernization (Navigation Rail & Dashboard)

`assyManager`를 단순 테이블 뷰어에서 정교한 엔터프라이즈 데스크톱 환경으로 전환하기 위한 대규모 UI/UX 개편 작업을 완료했습니다.

## 🚀 주요 변경 사항

### 1. 내비게이션 아키텍처 혁신 (Navigation Rail)
- **수직 사이드바 도입**: 기존 `QTabWidget`의 수평 탭 구조를 제거하고 좌측 수직 `NavigationRail` 시스템으로 전면 개편했습니다. 이를 통해 데이터 영역의 가독성을 확보하고 확장성을 높였습니다.
- **스마트 아이콘 매핑**: 테이블 이름을 분석하여 반도체 도메인(Wafer, Pkg, Yield 등)에 적합한 아이콘을 자동으로 할당하는 지능형 로직을 적용했습니다.
- **UX 폴리싱**: 언더바(_) 자동 치환을 통한 가독성 높은 텍스트 줄바꿈, 사이드바 너비 최적화, 호버/선택 비주얼 피드백 강화를 완료했습니다.

### 2. 홈 대시보드 및 실시간 상태 모니터링
- **DashboardPanel**: 앱 진입 시 현재 시스템의 활성 테이블 수와 연결 상태를 한눈에 볼 수 있는 랜딩 페이지를 구축했습니다.
- **실시간 상태바**: WebSocket 연결 상태를 상시 표시하는 공유 상태바를 하단에 배치하여 서버와의 동기화 여부를 상시 확인할 수 있게 했습니다.

### 3. 설정 패널 및 데이터 영속화
- **SettingsPanel**: 사용자 아이디(Identity)와 서버 IP/Port를 UI에서 직접 구성할 수 있는 전용 설정 페이지를 추가했습니다.
- **JSON Config 연동**: `client_settings.json` 파일을 통해 사용자의 환경 설정이 앱 재시작 후에도 유지되도록 구현했습니다.

### 4. 테이블 수명 주기 관리 (Lifecycle)
- **Close Table**: 사이드바 항목 우클릭을 통해 불필요한 테이블 페이지를 닫고 관련 데이터 모델 리소스를 즉시 해제하는 기능을 구현했습니다.
- **레이아웃 표준화**: 모든 테이블의 인덱스 열 너비를 50px로 고정하여 화면 전환 시의 시각적 이질감을 제거했습니다.

## 🛠️ 수정 및 추가된 파일
- `client/main.py`: 중앙 스택 레이아웃 및 내비게이션 컨트롤러 리팩토링.
- `client/ui/navigation_rail.py` [NEW]: 수직 사이드바 컴포넌트.
- `client/ui/panel_dashboard.py` [NEW]: 대시보드 위젯.
- `client/ui/panel_settings.py` [NEW]: 설정 관리 위젯.
- `client/config.py`: JSON 기반 설정 로드/저장 로직 추가.
- `client/client_settings.json` [NEW]: 사용자 설정 데이터.

- 사이드바를 통한 화면 전환, 테이블 종료, 설정 저장 및 즉각 반영 루프가 정상 동작합니다.
