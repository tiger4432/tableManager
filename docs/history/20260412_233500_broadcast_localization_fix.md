# 프로젝트 이력: 실시간 브로드캐스트 현지 시간(Local) 무결성 강화 (2026-04-12)

## 1. 개요
실시간 셀 수정이나 업서트 시 발생하는 WebSocket 브로드캐스트 메시지 내의 타임스탬프가 로컬 변환 로직을 타지 않고 UTC(Naive) 상태로 전송되어, UI상에서 시간이 일시적으로 UTC로 회귀하는 현상을 해결함.

## 2. 상세 작업 내용

### 2.1 시간 변환 유틸리티 (`to_local_str`) 전역화
- `main.py` 내부의 로컬 함수였던 `to_local_str`을 모듈 수준의 전역 함수로 승격시켜 모든 엔드포인트에서 접근 가능하도록 리팩토링함.

### 2.2 브로드캐스트 레이어 전면 수정
- **단일 셀 수정 (`update_cell`)**: 브로드캐스트 메시지의 `updated_at` 필드에 `to_local_str` 적용.
- **일괄 셀 수정 (`update_cells_batch`)**: 각 업데이트 항목의 `updated_at` 필드에 `to_local_str` 적용.
- **업서트 (`upsert_row`)**: 신규 생성(`row_create`) 및 기존 수정(`batch_cell_update`) 메시지 생성 시 시스템 컬럼 데이터를 현지 시간 포맷으로 강제 직렬화함.

## 3. 최종 결과
- 기존 데이터 로딩뿐만 아니라, **실시간 수정이 발생하는 즉시** UI에 나타나는 시간 정보도 항상 KST(+9)를 유지하는 것을 확인람.
