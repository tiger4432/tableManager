# 2026-04-22: 마지막 행 도달 및 데이터 로드 상태 알림 기능 추가

## 1. 개요
사용자가 테이블 하단에 도달하거나 대량 데이터를 로드할 때, 현재 상태를 명확히 인지할 수 있도록 하단 상태바(Status Bar) 알림 기능을 구현함.

## 2. 변경 사항
- **모델 시그널 확장**: `ApiLazyTableModel`에 `status_message_requested(str)` 시그널을 추가하여 데이터 로드 상태를 UI에 전달할 수 있는 채널 확보.
- **알림 발생 로직**:
    - **데이터 끝 도달**: 1K 로드 시 더 이상 가져올 데이터가 없으면 "마지막 행입니다." 메시지 송출.
    - **로드 완료 안내**: 데이터 청크 로드 완료 시 "N행 추가 로드 완료." 메시지 송출.
- **UI 연합**: `MainWindow`에서 해당 시그널을 `statusBar().showMessage()`와 연동하여 3초간 휘발성 메시지로 표시.

## 3. 검증 결과
- 1K 로드 클릭 시 남은 데이터가 없으면 즉시 상태바에 "마지막 행입니다." 표시 확인.
- 스크롤 및 일괄 로드 시 로드된 행 수가 상태바에 실시간 안내됨을 확인.
