# 프로젝트 이력: CSV 데이터 익스포트(Export) 기능 구축 (2026-04-12)

## 1. 개요
Spotfire, Excel 등 외부 데이터 분석 도구와의 연동성을 높이기 위해 테이블의 전체 데이터를 CSV 형식으로 추출하는 기능을 구현함.

## 2. 상세 작업 내용

### 2.1 서버 측 스트리밍 CSV 엔드포인트 (`/export`)
- **고성능 처리**: `StreamingResponse`와 `csv.writer`를 활용하여 수만 건의 데이터도 메모리 부하 없이 점진적으로 전송함.
- **데이터 평면화 (Flattening)**: JSON 구조의 `data` 컬럼을 개별 열로 펼치고, `created_at`, `updated_at` 등 시스템 컬럼을 통합하여 분석에 용이한 형태로 제공함.
- **호환성**: Excel에서 한글이 깨지지 않도록 UTF-8 BOM(\ufeff)을 추가함.

### 2.2 클라이언트 UI 통합
- **접점 마련**: 각 테이블의 `FilterToolBar`에 "📥 CSV 추출" 버튼을 추가하여 접근성을 확보함.
- **사용자 경험**: `QFileDialog`를 연동하여 사용자가 원하는 위치에 원하는 이름으로 파일을 저장할 수 있도록 함.
- **비동기 다운로드**: 다운로드 성공 시 안내 메시지를, 실패 시 상세 오류 메시지를 팝업으로 제공함.

## 3. 최종 결과
- 'CSV 추출' 버튼 클릭 시 현재 탭의 데이터 전량이 CSV로 즉각 다운로드되며, Excel 및 텍스트 에디터에서 정합성이 유지됨을 확인함.
