## 3. 구현 상세

### A. `+` 버튼 위치 이동 및 개선
- **기존**: `QTabWidget`의 코너 위젯 (가시성 낮음)
- **변경**: `FilterToolBar` 내부 (상단 툴바 우측 끝)
- **상세**:
    - 버튼 텍스트를 `+ 새 탭`으로 명확화.
    - `addTabRequested` 시그널을 통해 `MainWindow`에서 처리하도록 캡슐화.
    - Catppuccin 스타일 적용 (녹색 배경, 호버링 효과).

### B. 행 삭제 UI (`ExcelTableView`)
- **컨텍스트 메뉴**: 우클릭 시 "🗑️ 선택된 행 삭제" 항목 표시.
- **단축키**: `Delete` 키를 통한 삭제 트리거.
- **확인 절차**: `QMessageBox.question`을 통해 사용자 삭제 의사를 재확인하여 실수 방지.
- **동작**: 확인 시 서버에 `DELETE` 요청을 각 선택된 행의 `row_id`별로 전송.

### C. 모델 로컬 갱신 및 동기화 (`ApiLazyTableModel`)
- **WebSocket 연동**: 서버의 `row_delete` 브로드캐스트 이벤트 수신 처리.
- **데이터 갱신**:
    - `row_id` 기반으로 현재 로드된 데이터에서 인덱스 검색.
    - `beginRemoveRows` / `endRemoveRows`를 사용하여 UI에 삭제 애니메이션/갱신 반영.
    - `_total_count` 차감하여 전체 행 수 동기화.
